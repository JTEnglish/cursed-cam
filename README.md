# Cursed Cam
### Command Line Video Chat App
## Required Python Modules
* Socket
* Curses
* Pickle
* Numpy
* OpenCV
* Threading

## Runnnig the App:
```
python3 cursed-cam-udp.py <IP Address> <Port>
```
